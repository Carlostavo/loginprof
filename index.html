Cómo funciona

  • Esta solución NO requiere API Key - usa un método alternativo para acceder a las imágenes
  • Solo funciona con carpetas que estén configuradas como "Cualquier persona con el enlace"
  • Las imágenes deben estar directamente en la carpeta (no en subcarpetas)
  • Si agregas nuevas imágenes a la carpeta, haz clic en "Cargar Imágenes" para actualizar
  • El auto-desplazamiento se puede activar/desactivar en cualquier momento
  • Puedes arrastrar horizontalmente con el mouse para navegar manualmente
